基本介紹

php支援許多功能，可以輸出HTML的元素，可以使用通信技術做出搜尋列功能，亦可以撰寫按鈕功能使頁面可以與資料庫互動，但是任務必須按照順序執行(易於阻塞)

LOOPS

與許多語言一樣，有著:
For迴圈(針對對象，經常搭配計數器i++)
Do...while/while迴圈(針對條件或布林值做重複執行)
if迴圈(針對條件或布林值做單次執行)
foreach迴圈(適用於array)

ARRAY

陣列可適用於表單資料的收集，以及多維陣列的資料存放
方式 ，搭配http使其可以將收集到的資料傳遞到資料庫中
或者達成頁面與頁面間的傳遞
，經常性的搭配implode或者explode指令已完成其目的
，以避免語法錯誤

資料庫互動fetch

php支援各種mysql的語法，以及fetch等資料傳遞的方式 ，可以將create, delete, insert into等等的方法宣告為一個變數，在將其變數放進function作為執行的動作

AJAX

可以透過ajax.get post等等功能來抓取特定的資料，getelementbyid/name/className等等等

Function以及ECHO

經常性透過function傳入參數，驅動loops，然後return或者是echo特定資料 ，比如透過資料庫互動fetch所抓取到的迴圈搭配foreach，藉此將欄位輸出到頁面上

基本介紹以及Terminal

laravel串聯了Vue.js並形成了MVC架構，經常使用composer, artisan, npm三種類型的終端機指令執行 ，並且可以利用vue.js來解決同步執行的阻塞問題

Route

選擇controller的執行方式，可以是輸出頁面
或者執行某個model中的函式

Database

在其中設定表單欄位

Model

在其中設定表單欄位的屬性

View

所有的頁面html元素以及作為component的Vue.js功能撰寫

Controller

作為頁面間的中繼，可以使用index返回view頁面
使用create、store等函式對特定model的表單做CRUD
也可以自行撰寫方法來搭配vue檔案的執行

MVVM

Vue.js

    • v-on

      • 與按鈕搭配，監聽事件的觸發來形成效果
      • 與屬性作搭配，設定觸發時機等等
      • 與修飾符搭配，可設定觸發對象或次數等等
      Vue.js

    • 常見用法

      • Data函式存放可被template呼叫的變數
      • 使用methods設定生命週期間執行的函式
      • 使用props陣列接收父元件傳入的資料
      Vue.js

    • v-model

      • 可與data內的資料綁定
      • 可與input, checkbox, 下拉式選單標籤搭配
      • 可與修飾符.lazy .number .trim搭配
      Vue.js

    • methods

      • 設定axios獲取特定路徑的資料
      • 設定emit/on達成元件之間的傳接收資料
      • 使用async await達成非同步處理的動作
      Vue.js

    • v-for

      • 搭配 v-model或陣列可形成foreach迴圈
      • 搭配 :key屬性值可形成頁面元素的排序
      • 搭配 v-if的設定條件可行成頁面元素的篩選
      Vue.js

    • 生命週期

      • Vue可透過函式設定作用的期間
      • created, mounted, updated, destroyed
      • 在其中輸入要執行的函式或方法
      Vue.js
